## Maven Quickstart
1. Build the plugin
```console
git clone https://github.com/BlueWaveTechnologies/bluewave-doc-analysis-plugin.git
cd bluewave-doc-analysis-plugin
mvn install
```
2. Copy the `dist` directory into the bluewave plugins folder
